# ToDo iOS

ToDo project where you can create, modify, delete and list all the notes made in swiftui, using mvvm and applying testing.  

Features applied in the project:

- Swift
- SwiftUI
- SwiftData
- MVVM
- Unit Test
- Snapshot Testing
